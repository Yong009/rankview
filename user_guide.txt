# RankView 사용자 가이드 (User Guide)

본 문서는 RankView 시스템을 사용하는 일반 사용자를 위한 안내서입니다.

---

## 1. 시작하기 (로그인)
- 시스템에 접속하면 로그인 화면이 나타납니다.
- 부여받은 아이디와 비밀번호를 입력하여 접속하세요.
- 로그인 후 메인 대시보드로 이동하며, 본인의 폴더 및 등록된 상품 목록을 확인할 수 있습니다.

---

## 2. 폴더 관리 (사이드바)
폴더 기능을 사용하여 상품(키워드)을 카테고리별로 체계적으로 관리할 수 있습니다.

- **폴더 추가**: 좌측 사이드바 상단의 '+' 버튼을 클릭하여 새로운 폴더를 생성합니다.
- **폴더 선택**: 폴더 목록에서 특정 폴더를 클릭하면 해당 폴더에 속한 상품만 우측 테이블에 표시됩니다.
- **전체 대시보드**: 상단의 'RankView' 로고나 '전체 대시보드' 메뉴를 클릭하면 모든 폴더의 상품을 한꺼번에 볼 수 있습니다.

---

## 3. 상품(키워드) 등록 및 관리
- **상품 등록**: 
  1. 우측 상단의 '상품 등록' 버튼을 클릭합니다.
  2. 모달 창에 키워드, MID(네이버 상품 번호), 스토어명, 상품 주소 등을 입력합니다.
  3. '등록하기' 버튼을 누르면 현재 선택된 폴더에 상품이 추가됩니다.
- **선택 삭제**: 삭제하고 싶은 상품의 체크박스를 선택한 후 '선택 삭제' 버튼을 클릭하여 관리 목록에서 제외할 수 있습니다.

---

## 4. 순위 확인 및 업데이트
- **순위 새로고침**: 
  - 대시보드 상단의 '순위 새로고침' 버튼을 클릭하면 현재 화면에 표시된 모든 키워드의 실시간 순위를 네이버 검색 API를 통해 업데이트합니다.
  - 업데이트 진행 상황은 화면 상단의 진행 바(Progress Bar)를 통해 확인할 수 있습니다.
- **데이터 확인**: 테이블의 각 날짜별(1일~31일) 셀에는 해당 날짜에 기록된 순위가 표시됩니다.

---

## 5. 검색 및 필터링
- **검색**: 우측 상단의 검색창에 MID, 메모, 또는 스토어명을 입력하여 특정 상품을 빠르게 찾을 수 있습니다.
- **초기화**: 검색 필터를 해제하고 전체 목록을 다시 보려면 '초기화' 버튼을 클릭하세요.

---

## 6. 기타 부가 기능
- **엑셀 업로드**: 대량의 키워드를 한꺼번에 등록하고 싶을 때 '엑셀 업로드' 버튼을 사용합니다. (지정된 양식의 엑셀 파일이 필요합니다.)
- **카카오톡 문의**: 사용 중 문의사항이 있을 경우 우측 상단의 '카카오톡 문의하기' 버튼을 클릭하여 지원을 받을 수 있습니다.
- **네이버 API 설정**: API 연동 설정이 필요한 경우 '네이버 API 입력' 버튼을 통해 관리자로부터 전달받은 키워드를 설정하세요.

---

## 7. 주요 상태 표시
- **무료체험/유료회원**: 사용자 이름 옆에 현재 계정의 상태가 표시됩니다.
- **순위 변화**: 전일 대비 순위가 상승하거나 하락했을 경우 해당 수치가 테이블에 강조 표시될 수 있습니다.
